# 📝 增强版 Markdown 编辑器功能说明

## 🎯 新增功能概览

### ✨ 可视化编辑器
- **CodeMirror 集成** - 专业的代码编辑器，支持语法高亮
- **实时预览** - 分屏显示，Markdown 和 HTML 同步预览
- **智能缩进** - 自动处理列表和代码块缩进

### 🛠️ Markdown 工具栏
- **格式按钮** - 一键插入粗体、斜体、代码等格式
- **列表工具** - 快速创建有序和无序列表
- **链接插入** - 智能链接对话框
- **引用和分隔线** - 快速插入引用和分隔线

### ⌨️ 快捷键支持
- **Ctrl+B** - 粗体格式
- **Ctrl+I** - 斜体格式  
- **Ctrl+K** - 插入链接
- **Ctrl+S** - 保存文档
- **Enter** - 智能列表续行

### 🎨 编辑模式
- **分屏模式** - 编辑和预览同时显示
- **编辑模式** - 专注于 Markdown 编写
- **预览模式** - 专注于查看渲染结果

## 📋 支持的 Markdown 语法

### 🔤 文本格式
```markdown
**粗体文本**
*斜体文本*
`行内代码`
```

### 🔗 链接和图片
```markdown
[链接文本](https://example.com)
![图片描述](图片地址)
```

### 📝 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 📋 列表
```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

### 💬 引用和分隔线
```markdown
> 这是一个引用

---
```

## 🎮 使用方法

### 1. 启动编辑器
双击 `start-gui.bat` 启动服务器，浏览器访问 http://localhost:8899

### 2. 选择语言
在左侧面板点击要编辑的语言

### 3. 编辑内容
- 点击内容块进入编辑界面
- 使用工具栏按钮或快捷键添加格式
- 右侧实时预览 HTML 效果

### 4. 模式切换
- **分屏** - 同时查看编辑和预览
- **编辑** - 专注编写 Markdown
- **预览** - 查看最终效果

### 5. 保存修改
按 `Ctrl+S` 或点击顶部"保存"按钮
