# 快速使用指南

## 启动编辑器

**最简单的方法：**
双击运行 `start-editor.bat` 文件

**或者在 PowerShell 中运行：**
```powershell
.\start-editor.ps1
```

## 常用操作流程

### 1. 编辑现有语言内容
1. 启动编辑器
2. 选择 `2. 编辑指定语言`
3. 输入语言代码（如：zh-cn, en, ja）
4. 选择要编辑的条目
5. 输入新的标题和内容（支持 Markdown）
6. 选择 `0. 退出` 并保存

### 2. 添加新语言
1. 启动编辑器
2. 选择 `3. 添加新语言`
3. 输入新语言代码（如：fr, de, es）
4. 使用 `2. 编辑指定语言` 修改内容
5. 保存退出

### 3. 从 Markdown 文件导入
1. 准备 Markdown 文件（参考 example-about.md）
2. 启动编辑器
3. 选择 `5. 从 Markdown 导入内容`
4. 输入 Markdown 文件路径
5. 输入目标语言代码
6. 保存退出

## Markdown 格式示例

```markdown
## 标题1
这是第一段内容，支持**粗体**和*斜体*。

## 标题2
这是第二段内容，支持[链接](https://example.com)。
```

## 注意事项

- 修改会自动备份原文件
- 支持的语言：zh-cn, zh-tw, zh-hk, en, ja
- 退出时记得保存修改
- 出错时可以从备份恢复
