# 🚀 About-lang.js 增强版 Web GUI 编辑器

## 一键启动
双击运行：`start-gui.bat`

## 访问地址
浏览器打开：http://localhost:8899

## ✨ 主要功能
- ✅ **专业 Markdown 编辑器** - CodeMirror 语法高亮
- ✅ **实时分屏预览** - 编辑和预览同步显示
- ✅ **可视化工具栏** - 一键插入 Markdown 语法
- ✅ **智能快捷键** - Ctrl+B/I/K 等快速格式化
- ✅ **多种编辑模式** - 分屏/编辑/预览模式切换
- ✅ **字数统计** - 实时显示字符和词数
- ✅ **自动备份** - 每次保存前自动备份
- ✅ **文件导入导出** - 支持 Markdown 格式
- ✅ **响应式界面** - 桌面和移动设备适配

## 🎯 快速操作
1. **编辑内容**：点击左侧语言 → 右侧专业编辑器
2. **格式化文本**：使用工具栏或快捷键 (Ctrl+B粗体, Ctrl+I斜体)
3. **插入链接**：Ctrl+K 或点击链接按钮
4. **切换模式**：分屏/编辑/预览模式一键切换
5. **保存修改**：Ctrl+S 或点击顶部"💾 保存"

## 🛠️ Markdown 工具栏
- **📝 格式** - 粗体、斜体、代码
- **🔗 链接** - 智能链接插入
- **📋 列表** - 有序、无序列表
- **💬 其他** - 引用、分隔线

## ⌨️ 快捷键
- **Ctrl+S** - 保存
- **Ctrl+B** - 粗体
- **Ctrl+I** - 斜体  
- **Ctrl+K** - 插入链接

## 停止服务器
在控制台按 `Ctrl + C`
